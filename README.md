## 🎉 Let's go!

Para iniciar o projeto insira o comando <code>npm run dev</code> ou visite a **URL**: https://63c92b24931ef7786db4c8f1--meek-parfait-a81d1d.netlify.app/

## 💻 Projeto

Uma aplicação que utiliza da API do GitHub para buscar issues de um repositório, dados do perfil e exibir elas como um blog.


## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- Typescript
- React
- Node e NPM
- Vite


## 📚 Bibliotecas

- Styled Components
- React Hook Forms
- Date Fns
- Axios
- React Markdown
- Zod
